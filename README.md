# Blockchain Tracker (No IoT) - Full Project

This repository is a complete scaffold for a QR-code / event-based supply-chain tracker using Ethereum smart contracts (Solidity), Hardhat, and a React frontend. It includes:
- Solidity smart contract (contracts/Tracker.sol)
- Hardhat scripts and tests
- React frontend with html5-qrcode scanner integrated, ABI file, and a small UI
- Event-watcher Node.js service to listen for ItemUpdated events
- QR generator scripts
...
